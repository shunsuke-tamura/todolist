# todolist
vueをDockerで動かす  
1. Dockerでnode環境を立ち上げる
2. `npm install @vue/cli` を実行してvueをinstall
3. `vue create <名前>` を実行する
4. 色々設定する
5. 何もかも作られる(ディレクトリ構造、.gitignore、package.json等)
